Lead Project

Healthcare Revenue Cycle Risk Prediction

Machine Learning Framework Using Synthetic EHR Data

Developed ML framework predicting patient-level financial risk in healthcare settings, achieving 9.4× improvement over baseline in identifying high-risk patients. Demonstrates end-to-end ML pipeline development: data preprocessing, feature engineering, model training/evaluation, and interpretation via permutation feature importance.

Key Finding: Comorbidity burden emerged as the only reliable predictor (permutation importance: 0.062), while assumed socioeconomic factors showed minimal contribution.
Methods: Random Forest classification, stratified cross-validation, imbalanced learning techniques, synthetic data generation (Synthea)
Impact: Demonstrates how routine EHR documentation can enable proactive financial assistance workflows without additional data collection burden

Differential Expression Analysis of Tumor Tissue

Statistical Analysis of RNA-seq Data in R

Statistical analysis of RNA-sequencing data identifying genes differentially expressed between tumor and normal tissue samples. Demonstrates rigorous bioinformatics workflow: library size normalization, quality control via PCA, negative binomial regression for count data, and multiple testing correction.

Key Methods: Principal Component Analysis revealed 30.8% variance captured in PC1, clearly separating tumor from normal samples along the primary discriminant axis
Statistical Rigor: Negative binomial regression chosen over Poisson based on observed overdispersion; Benjamini-Hochberg FDR correction for multiple testing
