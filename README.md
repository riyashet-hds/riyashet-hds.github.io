# riyashet-hds.github.io

Source for my personal portfolio site: https://riyashet-hds.github.io

A minimal single-page site built with semantic HTML and CSS (Inter font, no frameworks). It summarises my
work in health data science, machine learning, and explainable AI.

## Structure

- `index.html`: page content (About, Selected Work, Skills, Contact)
- `styles.css`: styling; the accent colour is set in the `:root` block

## Run locally

Open `index.html` in a browser.

## Deploy

Commit and push to the `main` branch. GitHub Pages serves the site at the live URL within a couple of
minutes.
